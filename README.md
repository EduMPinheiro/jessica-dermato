# Dra. Jéssica Alencar - Dermatologia Clínica e Estética

Site premium desenvolvido para apresentação de serviços de dermatologia clínica, estética e tecnologias a laser. O projeto foca em transmitir confiança, elegância e profissionalismo através de um design minimalista e moderno.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização responsiva, custom properties (variáveis de cor e tipografia) e animações suaves, utilizando Vanilla CSS.
* **JavaScript:** Interatividade de interface, menu mobile responsivo e efeitos de *scroll reveal* utilizando a API `IntersectionObserver`.

## Características Visuais

* Design *clean* e minimalista, focado em tipografia refinada (combinação de Playfair Display e Inter).
* Paleta de cores sofisticada com tons suaves (off-white, verde sálvia, bege e dourado).
* Ícones vetoriais integrados para melhor performance e nitidez.
* Microinterações e animações de interface para uma experiência de usuário *premium*.
